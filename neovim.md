### Navigation/Cursor Movement


```
 k
h l
 j
```

`h` -> move cursor left  
`l` -> move cursor right  
`k` -> move cursor up  
`j` -> move cursor down  
***
`0` -> jump to the start of the line

***

`$` -> jump to the end of the line

`gg` -> jump to the start of the document
`G` -> jump to the end of the document






`Ctrl + ws ` - split window
